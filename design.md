# 程序员皇历助手 - 设计文档

## 应用概述

一款专为上班族(特别是程序员)打造的每日运势指导应用,结合中国传统皇历、星座运势、天气信息,提供个性化的穿搭建议、工作位置推荐、注意事项等实用指导。

## 设计理念

- **移动优先**: 竖屏单手操作,9:16比例设计
- **iOS原生感**: 遵循Apple Human Interface Guidelines
- **信息密度**: 首屏展示核心信息,减少滑动
- **视觉层次**: 清晰的信息分组和优先级

## 配色方案

- **主色调**: 紫色系(#7C3AED) - 神秘、智慧、科技感
- **辅助色**: 橙色(#F59E0B) - 活力、警示
- **背景色**: 
  - 浅色模式: 纯白(#FFFFFF)
  - 深色模式: 深灰(#151718)
- **卡片背景**: 
  - 浅色模式: 浅灰(#F5F5F5)
  - 深色模式: 深灰(#1E2022)

## 屏幕列表

### 1. 今日运势 (Home Screen)
**主要内容**:
- 日期显示(农历+公历)
- 今日宜忌(皇历)
- 综合运势评分(五星制)
- 天气信息卡片
- 穿搭建议卡片
- 工作位置建议卡片
- 程序员特色建议(编程语言运势、代码运势)

**布局**:
- 顶部: 日期+农历信息
- 中部: 运势评分+宜忌
- 下部: 滚动卡片列表(天气、穿搭、工作位置、程序员建议)

### 2. 详细运势 (Fortune Detail Screen)
**主要内容**:
- 五行运势详解
- 时辰吉凶
- 财运/事业运/健康运/感情运
- 幸运颜色/数字/方位

**布局**:
- 顶部: 返回按钮+标题
- 中部: 可滚动的详细运势卡片

### 3. 历史记录 (History Screen)
**主要内容**:
- 日历视图
- 历史运势查看
- 运势趋势图表

**布局**:
- Tab Bar图标: 日历图标
- 顶部: 月份选择器
- 中部: 日历网格
- 底部: 选中日期的简要信息

### 4. 设置 (Settings Screen)
**主要内容**:
- 通知设置(每日提醒时间)
- 主题切换(浅色/深色/跟随系统)
- 关于应用

**布局**:
- Tab Bar图标: 齿轮图标
- 分组列表样式

## 关键用户流程

### 主流程: 查看今日运势
1. 用户打开应用 → 自动显示今日运势首屏
2. 用户向下滚动 → 查看各类建议卡片
3. 用户点击"查看详细运势" → 进入详细运势页面
4. 用户返回首屏

### 次要流程: 查看历史运势
1. 用户点击Tab Bar"历史记录" → 进入历史页面
2. 用户选择日期 → 显示该日运势
3. 用户返回今日

### 设置流程
1. 用户点击Tab Bar"设置" → 进入设置页面
2. 用户调整通知时间/主题 → 自动保存
3. 用户返回首屏

## 程序员特色功能

### 1. 编程语言运势
- 每日推荐使用的编程语言(Python/JavaScript/Java/Go等)
- 显示该语言的"幸运指数"

### 2. 代码运势
- 今日适合写什么类型的代码(前端/后端/算法/重构)
- Bug出现概率预测

### 3. 工作建议
- 最佳工作时段
- 推荐工作位置(靠窗/角落/开放区)
- 是否适合开会/Code Review

### 4. 技术决策
- 今日是否适合上线部署
- 是否适合重构代码
- 是否适合学习新技术

## 数据来源

- **皇历数据**: 使用算法生成或API获取
- **天气数据**: 集成天气API
- **运势算法**: 基于日期、星座、五行的综合算法
- **程序员建议**: 基于日期哈希的随机算法

## 技术实现要点

- 使用AsyncStorage本地存储用户设置
- 使用React Native Reanimated实现流畅动画
- 卡片采用阴影和圆角设计
- 支持下拉刷新
- 支持深色模式自动切换
